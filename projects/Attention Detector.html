Attention Monitoring Detector

An interactive web platform utilizing real-time computer vision to monitor user engagement levels and head gaze direction.

Project Overview

The Attention Monitoring Detector addresses the challenge of maintaining focus during remote learning and working sessions. Leveraging webcam-based video streams, the app processes facial coordinates client-side to detect gaze shifts and head rotations. It will then classify attention states (e.g., focused, distracted) and visualize them in an interactive dashboard, calculating an active "attention score" dynamically.

Key Features

  • Webcam Head Tracking: Real-time head pose estimation without requiring external hardware calibrators.
  • Interactive Dashboard: High-performance graphing showing session-wide attention distributions.
  • Privacy-First Processing: All video frames are processed locally inside the user's browser, preventing server transmission of sensitive data.

ML Technologies

  • Computer Vision Pipelines: Facial landmark detection using MediaPipe to accurately capture head posiitions and gaze vectors.
  • Classification Model: A custom Convolutional Neural Network (CNN) trained to classify head orientations (Left, Right, Center, Down).
  • WebAssembly Acceleration: Model weights executed inside the browser using TensorFlow.js for low-latency client inference.

Learning Outcomes

Strengthened skills in edge computing and lightweight machine learning architecture. Designed robust responsive charts and handled browser permissions securely for webcam streams, optimizing frames-per-second processing for a smooth UI experience.
