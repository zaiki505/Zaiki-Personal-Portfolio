Console Application & Full-Stack System

House Rental Management System

A robust terminal-based rental application connecting landlords, property managers, and tenants with a secure MySQL database backend.

Project Overview

Developed as a semester project, this system was designed to digitize manual property booking, renting, and management workflows. By using C++ to build a modular command-line UI and connecting it to a MySQL relational database, the system ensures real-time updates, security checks, and persistent data storage across multiple user roles.

Key Features

  • Role-Based Portals: Distinct workflows for Landlords (managing listings), Tenants (searching and renting), and Representatives (administrative validation).
  • Real-Time Database Sync: Uses MySQL Connector/C++ for dynamic queries, prepared statements, and transactional safety.
  • Tenancy Agreements & Receipts: Automatically creates rental contracts and calculates booking fees, including discounts.
  • Robust Input Validation: Advanced string parsing and input sanitation routines to prevent buffer overflows and SQL injection.

Technologies & Concepts

  • Object-Oriented Programming (OOP): Classes, inheritance, encapsulation, and composition to represent landlords, tenants, and properties cleanly.
  • Relational DB Normalization: Database schema normalized up to 3NF to avoid redundancy and enforce referential integrity.
  • SQL Query Optimization: Structured joins, transaction handling, and optimized indexes to minimize latency.

Learning Outcomes

Gained deep proficiency in database-application integration. Learned to design normalized schemas, handle connection pools, build robust error-handling mechanisms for terminal-based user inputs, and coordinate project files in Git.
