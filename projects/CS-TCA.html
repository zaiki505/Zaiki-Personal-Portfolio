CS Training Centre Application System

A console-based workflow automator developed in C++ to handle student registration, billing, and scheduling at a regional IT academy.

Project Overview

The CS Training Centre Application (CS-TCA) digitizes regional course signups and class scheduling management. Using structured C++ programming without reliance on external SQL drivers, the system implements modular header configurations to process multi-student selections, compute regional fee discounts, and log bookings safely.

Role
Core Backend & Validation Designer
Tools
C++, Modular Programming, Structures & Arrays, Input Validation
Duration
UiTM Semester 1 (2023)

Key Features

  • Course Catalog Lookups: Instant lookup of schedules, durations, fees, and seat quotas for active IT training courses.
  • Course Registration validation: Validates user info, age limits, and prerequisite completion before enrollment.
  • Billing Calculations: Calculates tuition dynamically with automatic discount tiers for students booking multiple modules.
  • Text-Based Receipt Creator: Generates booking breakdowns in a clean format directly to the terminal console.

Technologies & Concepts

  • Modular Programming: Structured codebase with dedicated implementation (.cpp) and declaration (.h) files to optimize builds.
  • Data Collections: Employs structured arrays and data structs to keep student, registration, and course parameters organized.
  • Robust Console Input Filters: Filters keyboard inputs, catches type mismatches (such as inputs for numerical menus), and prevents runtime errors.

Learning Outcomes

Established foundational habits for structuring large-scale codebases modularly. Built robust text validation loops, managed internal application data structures, and practiced debugging variable states and scope in C++.
