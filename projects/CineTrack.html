CineTrack: A Smart Movie Tracker System

A CLI movie-tracking tool implementing custom pointers and linked list data structures to store, sort, and retrieve user watchlists dynamically.

Project Overview

CineTrack addresses the need for film enthusiasts to log, filter, and structure their watched/to-watch movie catalogs. Instead of using standard containers like standard vectors, this program builds a dynamic memory watchlist utilizing custom Linked Lists in C++, detailing pointer manipulation and sorting methodologies.

Role
Backend Developer & Data Structures Designer
Tools
C++, Custom Linked Lists, Searching & Sorting Algorithms, Git
Duration

Key Features

  • Dynamic Memory Watchlist: Allocates list nodes on-the-fly, allowing addition/deletion of entries without size limits.
  • Data Filtering: Integrated linear search algorithms to instantly filter movies by name, genre, rating or release year.
  • Watchlist Sorting: Bubble Sort algorithms sorting nodes alphabetically or by rating/release date.
  • Persistent Storage: Read/write modules that load and store watchlists in file formats, keeping data across executions.

Algorithms & Custom Structures

  • Pointer Manipulation: Created class node structs referencing next/previous node pointers to maintain node list connections.
  • Sorting Link Node Exchanges: Performed node pointer swaps directly rather than copy-assigning object data values, maximizing efficiency.
  • Resource Cleanups: Wrote custom destructors traversing the nodes recursively to deallocate dynamic memory buffers securely.

Learning Outcomes

Gained deep-level of understanding for dynamic memory allocation and pointer logic. Learned structural advantages of linked lists, implemented algorithmic sorting, and developed functional file stream pipelines for persistent console applications.
