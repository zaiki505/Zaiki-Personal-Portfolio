Network Design for Cyber Cat Café

A highly secure, segmented network architecture designed for a commercial entertainment venue hosting public Wi-Fi, gaming setups, and IoT devices.

Project Overview

Designed a comprehensive network topology for a fictional Cyber Cat Café. The project addresses concurrent connections for gaming consoles, POS machines, IoT surveillance cameras (for monitoring the cats), and high-bandwidth guest Wi-Fi, using VLSM subnetting and access-list policies for separation.
