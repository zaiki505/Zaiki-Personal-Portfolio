Network Architecture & Engineering

Network Design for Cyber Cat Café

A highly secure, segmented network architecture designed for a commercial entertainment venue hosting public Wi-Fi, gaming setups, and IoT devices.

Project Overview

Designed a comprehensive, scalable network topology for a fictional Cyber Cat Café. The project addresses concurrent connections for gaming consoles, POS machines, IoT surveillance cameras (for monitoring the cats), and high-bandwidth guest Wi-Fi, using VLSM subnetting and access-list policies for separation.

Role
Leader of the Project / Network Architect / Security Engineer
Tools
Packet Tracer, Subnetting (CIDR/VLSM), VLANs, IP Routing, DHCP, DNS
Duration
Group Project (2025)
Full Design
View Project Webpage

Network Topology & VLAN Setup

  • Virtual LAN Isolation: Created distinct logical subnets (VLANs) to prevent guests from accessing sensitive admin point-of-sale databases.
  • VLSM IP Allocation: Subnetted a Class C network space efficiently, allocating small pools for static routing links and larger blocks for clients.

Routing Protocols & Security

  • Dynamic Routing: Configured Open Shortest Path First (OSPF) to link multi-area routers and support rapid failover converges.
  • DHCP & DNS Services: Set up dedicated server nodes providing dynamic lease updates and local domain resolution services.

Learning Outcomes

Mastered enterprise-level IP subnet design and dynamic routing protocol deployments. Developed practical knowledge of VLAN tagging, access lists, and subnetting algorithms, optimizing traffic throughput for bandwidth-intensive environments.
